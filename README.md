
# Shopping Cart

A simple and user-friendly **Shopping Cart web application** built using **HTML, CSS, and JavaScript**.  
This project allows users to browse products, add items to the cart, manage orders, and proceed to checkout through a clean frontend interface.

## 📌 Features

- Product listing page
- Add to cart functionality
- View and manage cart items
- Checkout page
- User login and registration pages
- Admin page for management
- Responsive and simple UI design

## 📁 Project Structure

```bash
Shopping_Cart/
│── css/
│── js/
│── admin.html

````

## 🚀 How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/ravinga221/Shopping_Cart.git
```

### 2. Go to the project folder

```bash
cd Shopping_Cart
```

### 3. Open in browser

Open `index.html` in your browser.

You can also use **VS Code Live Server** for a better experience.

## 📄 Pages Included

* `index.html` – Home / product page
* `cart.html` – Shopping cart page
* `checkout.html` – Checkout page
* `login.html` – User login page
* `register.html` – User registration page
